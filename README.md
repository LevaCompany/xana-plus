# xana-x | xana eXtended
Это форк ЯП Xana от InnieSharp с моими ~~недо~~улучшениями <br>
Его синтаксис чем-то напоминает синтаксис INI-файла

## Hello, World
```ini
print "Hello, World"
```

Или, даже вот так:
```ini
[main]
hello = "Hello, World"
print hello
```

## Фичи
- Простой синтаксис (разберется даже ребенок)
- Много хаЦкерских функций (ого)
- И много багов...
